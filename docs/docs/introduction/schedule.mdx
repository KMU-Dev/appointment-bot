## 工作說明

我們的掛號對不隊將專案劃分成三個部分，分別由不同團隊指掌，以下簡要概述工作內容：

1. **Line Bot 組**：負責開發 Line Bot 伺服器，建構自然語言解析系統，將使用者訊息分類並交付業務邏輯處理。
    組員：**林子唐**、**趙子賢**
2. **Infermedica 組**：負責串接 Infermedica API，把 Line Bot 分類過的症狀資訊提供給 Infermedica API，得到相應的疾病資訊。
    組員：**吳宥緯**、**王晟彰**
3. **醫療機構組**：負責整理醫療院所資料，並依據使用者位置和疾病推算合適的醫療機構。
    組員：**劉芳瑜**、**林威良**
