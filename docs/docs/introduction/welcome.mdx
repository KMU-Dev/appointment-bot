非常感謝大家加入掛號對不隊這個家庭，想當初我們也都是被騙進來的（誤），還記得第一次討論時大家都不知道要做甚麼，
在芳瑜的提議下有了今天這個專案，一個人的力量或許很渺小，但當我們為了一個目標努力時，便能將夢想藍圖化作現實。

我始終相信一個好的團隊必須要有良好的溝通，前幾次開會時我太過心急，大部分的時間都是我在督促大家或是教大家一些工具，
但我真的很想聽聽你們的聲音，每個人都有獨特的人格特質和生命歷練，碰撞在一起時必然會激發燦爛的火花，
一個創新的想法能帶領我們到全新的格局，一句忠告能幫助我們離開錯誤的泥淖，誠心期待我們能有不同的想法，引領大家一同前行。

一艘船要能航行，需要訓練有素的舵手，也需要天氣的配合。
當你們緩緩升起船帆，航向未知的海域，我期望是那輕輕吹動的風浪，讓大家能展現自己的才能，勇敢地朝夢想航行。

<div style={{textAlign: 'right'}}>-- 組長 趙子賢</div>
