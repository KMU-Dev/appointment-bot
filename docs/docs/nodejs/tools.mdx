有許多開發工具可以幫助 JavaScript 的開發，其中最重要的就是整合是開發環境了 (IDE)，
現在就讓我們來安裝 JavaScript 開發最常用到的 Visual Studeio Code 吧！

## 安裝 Visual Studio Code

Visual Studeio Code 為 Microsoft 開發，讓使用者方便開發的整合是開發環境，原生支援 **JavaScript**、**TypeScript** 和 **Node.js**，
並可透過插件來支援 **C++**、**C#**、**Java**、**Python**、**PHP** 和 **Go** 的開發。

此外，Visual Studio Code 非常的清量，幾乎所有的功能都是靠插建來實現，壤使勇者能夠管理自己所需的功能，降低硬體效能的需求。

### 從官網安裝 Visual Studio Code

前往 [官方下載頁面](https://code.visualstudio.com/Download)，並依照不同的作業系統選取相應的安檔。


## 安裝必要插件

開發過程中，如果有好用的插件，將可以大大提升開發效能，並減少錯誤的發生，以下將介紹 2 個插件，這也是我們這次專案開發中會用到的。

### ESLint

<img 
    src="https://dbaeumer.gallerycdn.vsassets.io/extensions/dbaeumer/vscode-eslint/2.1.14/1607941542177/Microsoft.VisualStudio.Services.Icons.Default"
    alt="ESLint"
    width="128px"
    height="128px"
/>

ESLint 是 JavaScript 社群中最熱門的**靜態程式碼分析工具**，它可以幫助我們找到有問題的程式碼、修正部分錯誤，並讓程式碼保持統一與簡潔。
Visual Studio Code 作為最多人使用的 JavaScript IDE，自然有人提供相關的插件，讓我們在**程式碼**和下方的**狀態列**中，標註有問題的程式碼。

:::note
ESLint 插件運作需要安裝 ESLint 在**系統程式**或**專案**中，由於本專案的 `package.json` 配置檔中已經有將 ESLint 包含在內，
因此在執行 `yarn` 安裝相依套件時，便已將 ESLint 安裝在專案資料夾中了，僅須在下方狀態列的右下角，點擊 ESLint 按鈕允許其執行，
即可在專案中使用 ESLint 插件。
:::

下載位置：https://marketplace.visualstudio.com/items?itemName=dbaeumer.vscode-eslint


### Live Share Extension Pack

![Live Share Extension Pack](https://ms-vsliveshare.gallerycdn.vsassets.io/extensions/ms-vsliveshare/vsliveshare-pack/0.4.0/1591125099437/Microsoft.VisualStudio.Services.Icons.Default)

這個插件包中包含 Live Share 插件和 Live Share Audio 插件，您僅需安裝插件包即可同時安裝以上兩個插件。

Live Share 為 Microsoft 開發的插件，它讓使用者得以同步的編輯和測試程式碼，這讓我們在專案開發過程中發生問題時，能夠更方便的討論並解決問題。

:::note
首次使用時必須登入 Github 帳號，才能連線或開啟一個新的 Live Session。
:::

下載位置：https://marketplace.visualstudio.com/items?itemName=MS-vsliveshare.vsliveshare-pack

下載完成後，左邊的導覽列就會出現新的 Live Share 按鈕，點擊按鈕即可使用 Live Share 功能。
