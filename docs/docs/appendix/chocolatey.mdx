---
title: 安裝 Chocolatey (Windows)
---

Chocolatey 是 Windows 系統上的軟體管理工具，讓你能透過簡單的指令輕鬆的管理和升級軟體。

## 安裝 Chocolatey

以系統管理員身分開啟 PowerShell 並執行：

```powershell title="PowerShell"
Set-ExecutionPolicy Bypass -Scope Process -Force; [System.Net.ServicePointManager]::SecurityProtocol = [System.Net.ServicePointManager]::SecurityProtocol -bor 3072; iex ((New-Object System.Net.WebClient).DownloadString('https://chocolatey.org/install.ps1'))
```

安裝時如有詢問是否允許執行指令，請授予權限。

當指令執行完畢，您便可以開始使用 Chocolatey，如需完整的指令說明，請見 [Chocolatey 說明文件](https://docs.chocolatey.org/en-us/getting-started)。

## 安裝套件

安裝套件時，請以系統管理員身分開啟 PowerShell 並執行：

```powershell title="PowerShell"
choco install <package name>
```

舉例來說，如果要用 Chocolatey 安裝 Git，請以系統管理員身分開啟 PowerShell 並執行：

```powershell title="PowerShell"
choco install git
```

執行時，Chocolatey 會詢問你是否要允許執行安裝腳本，請輸入 **Y (Yes)**，或 **A (Accept all)**，如果想要省略此步驟，請在安裝時加上 `-y` 參數。

示範如下，請以系統管理員身分開啟 PowerShell 並執行：

```powershell title="PowerShell"
choco install git -y
```

Chocolatey 便會省略詢問步驟，直接執行安裝。
