在開始正式學習之前，我們先介紹 JavaScript 和 TypeScript 的不同，JavaScript 是完整的城使語言，而 TypeScript 充其量只是 JavaScript 的型別標註系統而已。
因此在往後的課程教學中，如果看到除了型別標註以外的語法，基本上都是 JavaScript 的語法，在這系列的教學中為了方便，我們並沒有嚴格區分兩者的不同，
但我還是希望大家能了解兩者在本質上是不同的東西，不應混為一談。

## 概述

在 JavaScript 中，程式是由敘述（statement）所組成，而每個敘述則用分號 `;` 隔開。

:::caution 警告
請注意這邊的分號是英文的分號喔，如果輸入中文的分號，將不具斷開 statement 的作用。
:::

如果一行內只包含一個 statement，則 `;` 可省略。

:::tip 提示
儘管分號可以省略，仍舊建議在每行的結尾加上分號以避免錯誤。
:::

此外，除了分隔變數、關鍵字或運算子的空白，其餘空白對於程式不具任何影響，因此可以利用空白來調整程式排版，使程式更易閱讀。


## 標準輸出

還記得上一章的 `index.ts` 嗎？裡頭包含了以下程式碼：

```typescript title="index.ts"
console.log('Hello world!');
```

其中 `console.log()` 能將括號內的內容顯示到螢幕上。

在這邊要特別注意的是，程式碼中的 Hello World 被單引號（`'`）夾住，這是 JavaScript 字串（string）常值（literal）的定義方式，
如果沒有被引號夾住，所有文字會被視為有意義的字元，包括變數名稱或關鍵字等等……。

此外，在 JavaScript 中可以透過 `+` 這個運算子連接兩字串，有關運算子的語法將會在後續章節介紹。

```typescript
'Hello' + ' ' + 'world!'  // Hello world!
```


## 變數

你可以將應用程式中用到的數值以變數（variable）來作為其象徵性的名字，而變數的名稱則稱為 identifer，根據不同的程式語言，identifer 有不同的命名規則。
JavaScript 定義變數規則如下：
- 必須以**英文字母**、**_** 或 **$** 作為變數名稱的第一個字
- 第二個字以後則可以包含**數字（0-9）**

:::tip 提示
儘管 JavaScript 的 identifer 符合上述規則程式便可正常執行，然而習慣上變數的名稱會以 **lower camel case** 的規則來命名，
其規則為變數的第一個單字為小寫，後續每個單字的第一個字為大寫。

**Example**：test、myName、lineBotName
:::

### 變數宣告關鍵字

JavaScript 宣告變數有兩種方式：
- 關鍵字 `var`，用來宣告 **局部變數（local）** 和 **全域變數（global）**，依照宣告的位置結果有所不同。
- 關鍵字 `let`，用來宣告 **區塊局部變數（block-scope local）**。

依照建議，除非有明確的用途需要宣告全域變數，否則強烈建議使用 `let` 作為宣告變數的方式。關於變數效力的教學，將會在下一章詳述。

變數宣告的語法定義：

```typescript
var identifer [= value];
let identifer [= value];
```

舉例如下：

```typescript
var x;
var y = 10;
let z;
let w = 'Hello';
```

### 變數值

如果我們未指定變數的值，或在變數宣告前就取用變數的值，那麼它就會擁有一個特殊的值 `undefined`，關於 `undefined` 的詳細說明，請見[資料型態](#資料型態)。

:::caution 警告
如果在變數宣告前就取用變數的值，且該變數是用 `let` 宣告，會丟出 `ReferenceError`，導致程式終止。
:::

```typescript
let x;
console.log('x 的值是 ' + x);  // x 的值是 undefined

console.log('y 的值是 ' + y);  // Uncaught ReferenceError: y is not defined
let y;
```

### 常數宣告

常數的命名規則和變數完全相同，變數效力則和 `let` 相同，唯一不同的點是宣告常數時使用的是 `const` 關鍵字，且常數的值須在宣告時指定，一經宣告即無法改變常數的值。

常數宣告的語法：

```typescript
const identifer = value;
```

舉例如下：

```typescript
const PI = 3.14;
```

對於不會變更的變數，建議以常數方式宣告，防止意外改變變數的值。

:::tip 提示
JavaScript 的常數名稱習慣以全大寫命名，單字間則以底線（_）間隔。

**Example**: MY_CONSTANT
:::

## 註解

註解是一段特殊的文字，在程式執行時會和空白一樣，被直譯器忽略。

### 單行註解

單行註解語法如下：

```typescript
// 單行註解
```

### 多行註解

多行註解可以用在任一行數的註解文字：

```typescript
/* 多行註解 */

/* 我可以
很多行 */

/*
 * 多行
 * 註解文件
 */
```

:::caution 警告
當使用多行註解語法時，不可在註解中再行註解，會導致 `SyntaxError`：

```typescript
/* 不可包含 /* 巢狀註解 */ SyntaxError */
```

以上註解會產生錯誤。
:::
